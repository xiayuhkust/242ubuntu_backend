# Twitter URL ID 提取服务

这是一个基于FastAPI的后端服务，用于从Excel文件中提取Twitter URL并获取对应的Twitter用户ID。

## 功能特点

- 支持Excel文件上传（.xlsx和.xls格式）
- 自动识别包含Twitter URL的列
- 提取Twitter用户名和ID
- 生成包含Twitter ID的新Excel文件
- 提供简单的Web界面进行操作
